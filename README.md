\# ArtifactRoom



Personal workspace for AI-generated artifacts and creative outputs.



This repository collects experiments, examples, reusable assets and practical knowledge created with AI tools such as ChatGPT, Gemini, Claude, AI Studio and other platforms.



\## Typical content



\- Images and graphics

\- Videos and animations

\- Vibe coding experiments

\- HTML prototypes and UI concepts

\- Documents and templates

\- Prompt collections

\- AI workflows and best practices

\- Notes, tips and reusable techniques



The goal is to build a growing library of useful AI-created assets and practical knowledge that can be reused across future projects.
